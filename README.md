|---------------|---------------------------------|

| %c            | Caracter                        |

| %p            | Puntero                         |
| %d            | Entero decimal firmado         |
| %i            | Entero                          |
| %u            | Entero sin signo               |
| %x            | Entero hexadecimal (minúsculas)|
| %X            | Entero hexadecimal (mayúsculas)|
| %%            | Solo el %                       |
